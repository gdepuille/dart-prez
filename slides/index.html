Concept fondamentaux

  • Tous ce qui est dans une variable est objet.
  • Un objet hérite de la classe Object
  • Tout le code est parsé au runtime.
  • Contrairement à Java il n'y a pas de mot clé public, protected et private
  • Deux mode d'éxécution :
    • Checked : Mode développeur pour avoir les erreurs de types et assert
    • Production: Optimisé pour la vitesse d'éxécution, mais ne réalise pas les contrôles de type ni les assert()
  • Toute variable non initialisée vaut l'objet null
  • Seul true est vrai
Objet = numbers, fonctions et même null.
Le préfix '_' signifie privé pour la librairie.

Type natif

  • numbers (int, double, num)
  • booleans (bool)
  • strings
  • lists (arrays)
  • maps
numbers : int et double (les deux hérite de num). Num contient les opérateurs de base et des fonctions (abs(), ceil(), floor()). Les autres sont dans une lib Math.
booleans : seul true est vrai. var name = 'Bob' if (name) // print in JS, not in Dart (in checked mode => exception is not a bool)
